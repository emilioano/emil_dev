TodayHelper is a Python project I built as a AI Developer student at Jensen.
Using different API's like open-meteo (coordinates and weather forecast), gemini and suno to generate custom feedback on some basic inputs.

Web front built with FastAPI:
Provide three inputs, and you will get back an customized AI generated response describing how you can get through the day in an optimal way based on your mood and the current weather at the specified location.
